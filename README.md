## Overview

A single-page, dark-themed developer portfolio built from scratch with plain HTML, CSS, and jQuery — no build step, no framework, no dependencies beyond a CDN-hosted jQuery and Google Fonts.

The centerpiece is a custom-animated **SVG infinity loop** in the hero section, representing the continuous REST API request/response cycle (`REQUEST → VALIDATE → PROCESS → RESPOND`) — a nod to the API-driven backend work described throughout the site.

## Sections

| Section | What's in it |

## Features

- 🎨 Fully custom design system (no CSS framework) — dark navy / teal / amber palette
- ♾️ Hand-built animated SVG infinity loop (SMIL `animateMotion`)
- 📜 Scroll-triggered reveal animations via `IntersectionObserver`
- 📊 Skill bars that animate into view on scroll
- 🧩 jQuery-driven accordions, mobile nav, and scroll-spy active links
- 📱 Fully responsive down to mobile
- ⚡ Zero build tools — open `index.html` and it just works

## Tech Stack

- HTML5 / CSS3 (custom properties, no framework)
- Vanilla JS + [jQuery 3.7](https://jquery.com/)
- SVG + SMIL animation
- Fonts: [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk), [Inter](https://fonts.google.com/specimen/Inter), [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono)

## Getting Started

Clone the repo and open the file directly — no install required:

```bash
git clone https://github.com/Ziad-Alsaghier/portfolio.git
cd portfolio
open index.html   # or just double-click it
```

### Deploying to GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, select the `main` branch and `/ (root)`.
4. Save — your site will be live at `https://<username>.github.io/<repo>/`.
